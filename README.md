# Digital Attendance Visualization Portal

## Overview
The Digital Attendance Visualization Portal is a web-based application designed to record, manage, and visualize attendance data for educational institutions. The system provides role-based access and graphical insights to improve attendance monitoring and decision-making.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: PostgreSQL
- Authentication: JWT
- Visualization: Chart.js / Recharts

## System Architecture
The application follows a PERN stack architecture where the React frontend communicates with an Express-based REST API, which interacts with a PostgreSQL database for persistent storage.
